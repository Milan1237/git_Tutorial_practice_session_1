git init : intializes the git in the current folder
git status : tells you status of the folder: wheather it has been added or not
git log : tells you the history or changes being made in the file

git stash : makes file in staging area to backstage so that it can't get commited or saved;
git stash pop : we can again take it to staging area by using this command
git stash clear: makes all the stash background file which there to be saved if needed (for example experimental file which a programmer wants to test but doesn't want to make a history out of it );
git remote add origin <url> : it decides where upload in the github
git push origin master: upload the files to github ;
